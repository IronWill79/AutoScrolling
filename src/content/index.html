Options | AutoScrolling
Scrolling speed:

The value is scrolling speed. Max is 99, and min is 1.

Scrolling stop by click:

The value decides the weather stoppling the scrolling when you click the scrolling window.

Stop scrolling when hovering over something:

Stop scrolling on hover

Shortcut - Toggle Autoscrolling:

Toggle the tab's scrolling by keyboard shortcut
The value should be a combination of keycodes combined with the '+' sign.
Ex: Ctrl+Shift+Insert.
Check out the link for the keycodes: Key combinations
PS: Some key combinations are reserved by the browser and can't be reassigned to this.
